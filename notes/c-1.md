1. inicializacao
2. verificacao
3. atualizacao

```c

}
```

## Function `main`

A main, tambem chamada de funcao principal, eh o entrypoint (ponto de entrada) do nosso codigo

```py
a = 5
b = 6

def soma(a, b):
  return a + b

print(soma(a, b))
```

```c
int a = 5, b = 6;

double soma(double a, double b) {
  return a + b;
}

int main() {
  printf("soma = %lf\n", soma(a, b));
  return 0;
}
```
